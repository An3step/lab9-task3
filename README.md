# Project Description

Приложение представляет собой функциональный аудиоплеер, способный сканировать файловую систему, формировать плейлисты и управлять воспроизведением треков. Реализована интеграция C++ бэкенда с QML фронтендом.

### Key Features
*   **Файловый менеджер:** Просмотр списка аудиофайлов из выбранной папки или медиатеки устройства (QDir, QFileSystemModel).
*   **Плеер:** Воспроизведение, пауза, остановка, перемотка треков (QtMultimedia).
*   **Метаданные:** Отображение обложки альбома (если встроена в файл).
*   **Плейлист:** Формирование очереди воспроизведения с помощью QML ListView.
*   **Локализация:** Перевод интерфейса на русский, английский и белорусский языки.

### Tech Stack
*   **Язык:** C++17, QML
*   **UI:** Qt Quick Controls 2
*   **Мультимедиа:** QtMultimedia (Audio, MediaPlayer)
*   **База данных:** SQLite (для хранения плейлистов и метаданных)
*   **Сборка:** CMake

---

## Platform-Specific UI Requirements

*   **Android:** Material Design, `Drawer` для навигации, `BottomNavigation` с крупными кнопками управления.
*   **iOS:** Плавные анимации переходов, `TabBar` внизу, минималистичный плеер в стиле Apple Music.
*   **Linux:** Классический десктопный интерфейс с верхним меню и панелью инструментов. Поддержка drag-and-drop файлов в окно.
*   **Web (WebAssembly):** Адаптивный интерфейс: на мобильных используется навигация как в Android, на десктопе — как в Linux. Поддержка хоткеев клавиатуры.

## How to Build

Для сборки требуются:
* **Qt 6.6+** с модулями `qtmultimedia`, `qtsql`, `qtwebassembly` (для Web).
* **CMake 3.20+**.
* **Android NDK** (для Android), **Xcode** (для iOS), **Emscripten** (для WebAssembly).
